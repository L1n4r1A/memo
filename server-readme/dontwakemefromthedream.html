サーバーについて

使い方

　みんなで楽しくやりましょう。

チャンネルについて

　メインとなるチャンネルは#textです。何か一個の話題(特定のゲームの話や小説の設定案など)はスレッドを立てるようにしてください。
　サーバーを美しく維持するための活動のご協力をよろしくお願い申し上げます。

認証(@Verified)について

　このサーバーではスパム防止の為、招待の際に認証を行っております。
　ルールを理解出来た方にロールを付与しますのでまぁ、お待ち下さい。

マネージャー(管理者)について

　このサーバーでは管理及び維持の為にマネージャーシステムを導入しております。
　@初級管理者 ＜ @中級管理者 ＜ @管理補佐 とランクが高くなります。

マネージャーの役割

　@初級管理者 ：サーバーの一部の管理を移譲しています。
　@中級管理者 ：サーバーの管理を担当しています。
　@管理補佐 ：サーバーの管理／運営を担当しています。
